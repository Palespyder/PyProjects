Usage

CSS helper contains short code (such as background, color, font size, box corner, etc) that can be used to support existing features.

Colors

  • *-lime
  • *-green
  • *-emerald
  • *-teal
  • *-cyan
  • *-cobalt
  • *-indigo
  • *-violet
  • *-pink
  • *-magenta
  • *-crimson
  • *-red
  • *-orange
  • *-amber
  • *-yellow
  • *-brown
  • *-olive
  • *-steel
  • *-mauve
  • *-taupe
  • *-black
  • *-silver
  • *-white

To set background color use prefix bg, for set text color use prefix color

<div class="bg-lime">...</div>
<span class="color-green">...</span>

To set border color use prefix border (if the object doesnt have border, add class bordered to add border)

Change border size with class border-2x or border-3x

<div class="border-red">...</div>

To set outline color for components use prefix outline, this work on hover.

<div class="outline-amber">...</div>
